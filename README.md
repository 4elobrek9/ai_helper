# 💎 LUMI AI — Project "Living Shard"

**LUMI AI** — это персональный голосовой ассистент, объединяющий эстетику **Glassmorphism** и мощь ИИ(( в новых апдейтах будет локальный. в начале он был, но это не для всех явно )) . Философия проекта: *"Твой голос — её разум"*. Это компактный, стильный и быстрый инструмент, который всегда под рукой, но не мешает твоей работе.

## 🎯 Смысл приложения

В отличие от громоздких интерфейсов, **LUMI** построена вокруг концепции **"Живого осколка" (Living Shard)**:
*   **Минимализм:** Окно размером **400x600** идеально вписывается в угол экрана.
*   **Визуальный отклик:** Интерактивные циановые волны реагируют на твой голос в реальном времени.
*   **Автономность:** Вы сами управляете своими API-ключами и данными, обеспечивая полную приватность.
*   **Удобство:** Приложение легко сворачивается в трей и вызывается одной фразой.

## 🔑 Как получить API ключи

Для полноценной работы LUMI необходимы два "органа чувств":

### 1. **Picovoice API** (Слух / Активация)
Используется для распознавания кодового слова (Wake Word), чтобы ассистент начал слушать вас.
*   Перейдите на сайт [**picovoice.ai**](https://picovoice.ai/).
*   Зарегистрируйтесь (бесплатно для разработчиков).
*   В консоли управления скопируйте ваш **Access Key**.

### 2. **Mistral API** (Мозг / Интеллект)
Используется для генерации ответов и общения.
*   Перейдите на портал [**console.mistral.ai**](https://console.mistral.ai/).
*   Создайте аккаунт и перейдите в раздел **API Keys**.
*   Создайте новый ключ и сохраните его.

> **Примечание:** Полученные ключи нужно вставить в разделе **SETTINGS** прямо внутри запущенного приложения.

## 🚀 Быстрая установка

Я максимально упростил процесс развертывания, чтобы вам не пришлось возиться с консолью.

1. **Скачайте проект:** Склонируйте репозиторий или скачайте ZIP-архив.
2. **Запустите установщик:** Найдите в корневой папке файл **`launch.bat`** и запустите его.
3. **Дождитесь магии:** Скрипт сам создаст виртуальное окружение (`.venv`), установит все нужные библиотеки (Flet, Picovoice, Mistral и др.) и запустит приложение.
